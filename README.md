# `pkuthesis`：信科 2025 年本科生毕业论文 LaTeX 文档类

## 特性

- 尽可能模仿 2025 年信科毕业论文 Word 模板；
- 改了一些字体；
- 提供最小的功能；
- 易于修改，作者不希望文档类充斥着 Plain TeX 或 LaTeX2e 古神语；
- 主要满足作者本人的需求。
